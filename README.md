-코드의 실행 스크린샷과 구현내용 설명

![실행화면](img/sc1.png)
- 구현한내용(위그림참조)
	- 체크박스를 활용하여 항목을 복수로 선택할 수 있도록 함.
	- 라디오버튼를 이용하여 항목을 단일로 선택할 수 있도록 함.

	- 초기화 버튼을 누르면 체크박스와 라디오버튼의 선택이 초기화되고, 리스트박스에 추가된 항목과 총 합계가 초기화됨.
